Projeto iZer: Colaborativo, ágil e acessível.

O sistema de gerenciamento de fretes que oferece vantagens à todos os envolvidos.

Diferenciais de Negócio:

  • Serviço 100% colaborativo
  • Preços justos para seus fretes
  • Oportunidade de ganho para todos os usuários
  • Simplicidade focando em eficiência
  • Aproveitamento de rotas, agilizando as entregas
  • Priorização de entregas de acordo com sua urgência
  • Sistema de metas e recompensas
  • Sistema de comparação de preços finais (frete e produto)

Diferenciais Tecnológicos:

  • API desenvolvida para integrações externas
  • Hospedado no Azure, nuvem da Microsoft
  • Disponível como WebApp e aplicativo Android
  • Mapas e rotas traçadas pela Google Maps Directions API
  • Consulta para comparações através da API do BuscaPé
  • Segurança nos pagamentos garantida pelo PagSeguro
  • Desenvolvimento focado em melhoria contínua

O Projeto iZer é um sistema de serviço colaborativo, ou seja, você pode participar!
